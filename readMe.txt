== Frequently Asked Questions ==

= What Do I Need To Use The Plugin =

1.	You need to have WooCommerce plugin installed and activated on your WordPress site.
2.	You need to have the interwswitch webpay woocommerce VTU plugin set up [Interswitch](https://github.com/makinde2013/wordpress_interswitch_webpay_woocommerce_vtu_plugin)
2.	You need to signup for MobileNig API on [MobileNig](https://mobilenig.com/register.php)
